2 - APIs & Auth -> APIs -> Enable "books" API

3 - APIs & Auth -> Credentials -> Create new Key: Server Key (ACCEPT REQUESTS FROM THESE SERVER IP ADDRESSES: leave empty)

4 - You have the API KEY
